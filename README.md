# Design-Pattern

1. Creational

  * 抽象工廠 (Abstract Factory)
  * 構造器 (Builder Pattern)
  * 工廠方法 (Factory Method pattern)
  * 原型 (Prototype pattern)
  * 單例模式 (Singleton pattern)

2. Structure

  * 配接器(Adapter pattern)
  * 橋接(Bridge pattern)
  * 組合(Composite pattern)
  * 裝飾(Decorator pattern)
  * 外觀(Façade pattern)
  * 享元(Flyweight pattern)
  * 代理(Proxy pattern)

3. Behavior

  * 職責鏈(Chain-of-responsibility pattern)
  * 命令(Command pattern)
  * 翻譯(Interpreter pattern)
  * 疊代器(Iterator pattern)
  * 中介者(Mediator pattern)
  * 回憶(Memento pattern)
  * 觀察者(Observer pattern)
  * 狀態機(State pattern)
  * 策略(Strategy pattern)
  * 模板方法(Template method pattern)
  * 參觀者(Visitor)
